# 📊 Sentiment Analysis & Emotion Detection using NLP

## Overview

This project performs Sentiment Analysis and Emotion Detection on book reviews using Natural Language Processing (NLP).

It cleans text data, classifies reviews into Positive, Negative, and Neutral sentiments, detects basic emotions, and visualizes the results with charts.

---

## Features

- Text Cleaning
- Stopword Removal
- Sentiment Analysis using VADER
- Emotion Detection
- Trend Analysis
- Data Visualization
- CSV Output Generation

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- NLTK (VADER)
